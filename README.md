# Music Copilot 🎵

An AI-powered music production assistant — currently in active development.

**The problem:** Music production tools are powerful but assume expert 
knowledge. The gap between musical intent and production execution is wide, 
and there's no real-time guide that bridges it.

**What I'm building:** A structured copilot that helps with music theory, 
Ableton workflow, sound engineering decisions, and eventually takes agentic 
actions directly inside Ableton via MCP.

## Current State
- PRD drafted (see `/docs`)
- Frontend: planned in Cursor
- Backend: knowledge base under construction
- Ableton MCP connection: Phase 2

## Why I'm Building This
I'm a Senior PM who also makes music. I took Ableton classes and kept 
hitting the same wall — too many options, not enough structure, no guide 
that meets you where you are. So I decided to build the thing I needed.

This is also how I learn new stacks. Building something real for a problem 
I actually have is faster and more honest than tutorials.
